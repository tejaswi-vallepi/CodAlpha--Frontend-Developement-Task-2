## Overview

The Advanced Calculator is a responsive web application built using HTML, CSS, and JavaScript. It provides a modern and user-friendly interface for performing basic arithmetic calculations with support for keyboard inputs and real-time expression evaluation.

This project demonstrates fundamental front-end development concepts, including responsive design, DOM manipulation, event handling, and interactive user interfaces.

## Features

### 1. Modern User Interface

* Clean and attractive calculator design.
* Gradient background for enhanced visual appearance.
* Rounded corners and shadow effects for a professional look.

### 2. Basic Arithmetic Operations

Supports the following operations:

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Percentage (%)

### 3. Responsive Design

* Works smoothly on desktops, tablets, and mobile devices.
* Layout automatically adjusts to different screen sizes.

### 4. Interactive Buttons

* Hover effects provide visual feedback.
* Well-organized button arrangement for easy usage.

### 5. Keyboard Support

Users can perform calculations directly using the keyboard:

| Key       | Function              |
| --------- | --------------------- |
| 0-9       | Number Input          |
| + - * / % | Operators             |
| Enter     | Calculate Result      |
| Backspace | Delete Last Character |
| Escape    | Clear Display         |

### 6. Clear and Delete Functions

* C button clears the entire expression.
* Backspace button removes the last entered character.

### 7. Error Handling

* Invalid expressions display an "Error" message.
* Prevents application crashes caused by incorrect inputs.

## How It Works

1. Users enter numbers and operators using buttons or keyboard inputs.
2. The entered expression is displayed on the calculator screen.
3. When the "=" button or Enter key is pressed, JavaScript evaluates the expression.
4. The result is displayed instantly.
5. Invalid inputs are handled through error detection mechanisms.
